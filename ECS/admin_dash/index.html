E-Complaint system is used to manage and forward complaints related any government field posted by users. User can register and make profile here and they get access to post complaint regarding any field. Further Admin will proceed with those complaints to higher authority to be resolved if admin genuinely think that the complaint given by the user is genuine or admin has to dismiss any unnecessary complaints. This site takes one step forward towards online services for government field and make resources easy to use.
